# 用户反馈补充：抖音链接具体格式要求

## 抖音视频账号链接格式
用户明确要求俱乐部详情页顶部的抖音账号跳转链接应使用抖音短链接格式：

**示例**：`https://v.douyin.com/79_RsP7TfOY/`

点击后直接访问该俱乐部的抖音视频账号首页。

## 详情页完整功能
以"星火电竞护航"为例，用户点击俱乐部卡片后：

**顶部区域**：
- 俱乐部名称和Logo
- 抖音短链接（如 https://v.douyin.com/xxx/），点击跳转到抖音视频账号首页

**中部价格展示**：
- 清晰列出该俱乐部的所有陪玩项目和对应价格
  - 基础护航价格
  - 跑刀价格
  - 代肝价格
  - 其他服务项目及价格

**底部**：
- 立即下单按钮（跳转到该俱乐部的官方下单链接）

## 数据字段补充
每个俱乐部需要包含：
- 抖音短链接（用于跳转视频账号首页）
- 下单跳转链接（用于跳转官方下单页面）
- 各类型陪玩价格（基础护航/跑刀/代肝等）
