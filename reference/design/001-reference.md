# 功能升级需求：俱乐部详情页

## 升级目标
在 delta-escort-club-rank 项目基础上，新增俱乐部详情页功能。

## 需求详细说明

### 1. 俱乐部列表页升级
- 每个俱乐部的卡片上标注各类型陪玩价格（基础护航价格、跑刀价格、代肝价格等）
- 点击俱乐部卡片时，**不再直接跳转下单链接**，而是打开俱乐部详情页

### 2. 新增俱乐部详情页
用户点击俱乐部后进入详情页，包含：

**顶部区域**：
- 俱乐部名称 + Logo
- 抖音账号跳转按钮（点击直接跳转到该俱乐部的抖音主页/视频账号）

**陪玩服务价格区**：
- 清晰展示该俱乐部提供的所有陪玩类型及其价格
  - 基础护航价格（单局）
  - 跑刀价格
  - 代肝价格
  - 其他服务价格
- 价格以卡片或列表形式清晰展示

**俱乐部信息区**：
- 粉丝量、好评率、月下单量、平均撤离率
- 主打服务标签
- 是否支持售后

**底部行动区**：
- 醒目的"立即下单"按钮，点击跳转俱乐部官方下单链接

### 3. 交互流程
```
用户点击俱乐部卡片
    ↓
打开俱乐部详情页（弹窗或新页面）
    ↓
查看陪玩类型和价格
    ↓
点击顶部抖音账号跳转，或点击底部立即下单
```

### 4. 设计风格
保持现有的硬核军事科技风格，深色调 + 战术绿色 + 网格纹理，契合《三角洲行动》游戏调性。

### 5. 技术要求
- 详情页可采用弹窗(modal)或独立路由实现
- 响应式移动端适配
- 预留飞书多维表格数据对接接口（当前使用模拟数据）
